Terminal.

 

La siguiente guía comprende la conexión con el servicio de timbrado utilizando Terminal o Linea de Comandos y cURL.

El código de esta guía se encuentra disponible para su descarga en el repositorio de ejemplos para Timbre Fiscal.

Generar Timbre

Para generar un timbre utilizando el servidor de pruebas es necesario tener los siguientes datos del servidor a la mano:

 Servidor de prueba: https://staging.diverza.com/stamp
 Token de prueba: ABCD1234
 Metodo HTTP: POST
 Header HTTP: x-auth-token

El siguiente paso es usando la terminal o linea de comandos colocarnos en la ruta donde se encuentra el archivo que creamos previamente y ejecutar lo siguiente:

cURL https://staging.diverza.com/stamp -X POST --data @cfdi.xml -H "x-auth-token: ABCD1234" -i

Nota: La dirección del ejecutable cURL puede variar entre sistemas operativos y procesos de instalación.

Si revisamos el comando prodremos ver que la petición es dirigida a la URL del servicio de timbrado, utilizando el metodo POST, agregando como cuerpo del mensaje el CFDI a timbrar y como header el token de seguridad, que en este caso es el que se utiliza para el ambiende de pruebas ABCD1234.
